# @cloudillo/calendar-ui

React calendar component — day, week, and month views with drag, resize, and touch support. Built for [Cloudillo](https://cloudillo.org) but framework-agnostic at the API level.

## Install

    npm install @cloudillo/calendar-ui react

## Usage

```ts
import '@cloudillo/calendar-ui/calendar-ui.css'
```

See the source for the current component API:

https://github.com/cloudillo/cloudillo/tree/main/libs/calendar-ui

## Peer dependencies

`react ^19.1.0`
