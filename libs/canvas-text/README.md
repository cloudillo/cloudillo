# @cloudillo/canvas-text

Rich text rendering, editing, and SVG export for canvas-based React apps — built on Quill and Yjs. Powers text objects in Cloudillo's canvas apps (prezillo, ideallo).

## Install

    npm install @cloudillo/canvas-text quill yjs y-quill react

## Peer dependencies

`react ^19.0.0`, `quill ^2.0.0`, `yjs ^13.6.0`, `y-quill ^0.1.0`
