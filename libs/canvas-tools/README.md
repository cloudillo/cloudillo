# @cloudillo/canvas-tools

SVG transform gizmos for React — rotation, pivot, and resize handles for building canvas editors. Used by Cloudillo's canvas-based apps (prezillo, ideallo).

## Install

    npm install @cloudillo/canvas-tools react react-svg-canvas

## Usage

See the source for the current export surface:

https://github.com/cloudillo/cloudillo/tree/main/libs/canvas-tools

CSS:

```ts
import '@cloudillo/canvas-tools/components.css'
```

## Peer dependencies

`react ^19.1.0`
