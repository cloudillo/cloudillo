# @cloudillo/crdt

Yjs CRDT collaborative editing client for [Cloudillo](https://cloudillo.org) — IndexedDB persistence, awareness, and WebSocket sync.

Used by Cloudillo apps that need real-time collaborative editing of text, canvas objects, or other CRDT-friendly data structures (quillo, prezillo, ideallo, notillo).

## Install

    npm install @cloudillo/crdt yjs

## Usage

The simplest path is via `useCloudilloEditor` from [`@cloudillo/react`](https://www.npmjs.com/package/@cloudillo/react). For non-React consumers, see the source:

https://github.com/cloudillo/cloudillo/tree/main/libs/crdt
